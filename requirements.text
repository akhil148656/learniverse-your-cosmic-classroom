REQUIREMENTS & SPECIFICATIONS — LEARNIVERSE
===========================================

System Requirements
-------------------
- OS: Windows, macOS, or Linux
- Node.js: 18+ recommended (Vite + modern tooling)
- npm: 9+ (bundled with Node)
- Git: latest stable
- Supabase CLI: recommended for migrations and Edge Functions

Runtime Requirements
--------------------
Frontend
- Vite (build/dev)
- React 18 + TypeScript
- Tailwind CSS + shadcn UI + Radix UI

Backend
- Supabase project (Postgres + Auth + RLS)
- Supabase Edge Functions (Deno runtime)

Required Environment Variables
------------------------------
Frontend (.env)
- VITE_SUPABASE_URL
- VITE_SUPABASE_PUBLISHABLE_KEY

Supabase Edge Function Secrets
Required for AI:
- GEMINI_API_KEY (Google AI Studio)
- Optional: GEMINI_MODEL (default: gemini-2.0-flash)

Optional AI Provider Fallback:
- GROQ_API_KEY
- Optional: GROQ_MODEL (default: llama-3.1-8b-instant)

Optional YouTube Search:
- YOUTUBE_API_KEY

Supabase Project Requirement
----------------------------
- Project ref: wonpmcjrkkuyoubwosfr
- Edge Functions enabled
- RLS enabled on all tables
